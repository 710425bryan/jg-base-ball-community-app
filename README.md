# jg-base-ball-community-app

中港熊戰 / 中港國小社區棒球管理系統。

## AI / Agent 工作入口

本專案的 AI 規則以 `AGENTS.md` 為準：

1. 先讀 `AGENTS.md`
2. 需要理解功能資料流時讀 `docs/PROJECT_LOGIC.md`
3. 需要定位檔案時讀 `docs/FILE_MAP.md`
4. 涉及 migration 時讀 `docs/MIGRATIONS.md`
5. 涉及 Edge Function 或環境變數時讀 `docs/EDGE_FUNCTIONS.md`
6. 依任務類型讀 `AI_SKILLS.md` 與 `.codex/skills/*/SKILL.md`

## 開發指令

- 開發：`pnpm dev`
- 建置：`pnpm build`
- 預覽：`pnpm preview`
